## ADDED Requirements

### Requirement: Repository README includes an introduction
The repository SHALL provide a top-level README introduction that explains the purpose of the workspace and gives new contributors an immediate starting point.

#### Scenario: Opening the repository for the first time
- **WHEN** a contributor opens the repository
- **THEN** they can read a short explanation of the repository purpose in `README.md`

### Requirement: Repository README includes next steps
The README SHALL list the first actions a contributor should take after opening the repository.

#### Scenario: Looking for how to begin
- **WHEN** a contributor finishes reading the introduction
- **THEN** the README shows concrete next steps such as reviewing structure or running the project
